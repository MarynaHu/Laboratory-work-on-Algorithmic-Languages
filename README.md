## LAB03: Functions as Objects, Lambda, Closures

The goal of this lab is to explore Python functions as first-class objects and understand how functions can:  
► be assigned to variables  
► be passed as arguments  
► be created dynamically  
► capture variables from their environment (closures)  

### Python version used
Python 3.14+

### Instructions on how to run the code
Activating a virtual environment via terminal:  
```
python -m venv .venv  
.venv\Scripts\activate  
pip install -r requirements.txt
```
Сommand to run the program:  
```
python ./Lab3.py
```
### Short description of the implemented tasks

The program prints six sections (A–F), each demonstrating: Functions as Objects, Sorting with Lambda, Function Factory, Closure Counter, Closure Counter, and Functional Composition.  
Detailed explanations for each section and answers to theoretical questions are in the file answers 3.md
