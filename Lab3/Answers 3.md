**Answers 3**

*1\. What does it mean that functions in Python are first-class objects?*  
It means that functions can be treated just like any other data type.

*2\. What is the difference between a function defined with def and a lambda expression?*  
A function defined with def has a name, can contain multiple lines of code, and requires an explicit return statement. A lambda expression creates an anonymous (unnamed) function that is restricted to a single expression and automatically returns the result of that expression.

*3\. What is a closure?*  
A closure is a nested (inner) function that "remembers" the variables from its enclosing (outer) function's scope, even after the outer function has finished executing.

*4\. In what situations are closures useful?*  
Closures are very useful for data hiding and encapsulation, creating function factories, and building decorators.
